# DreamAI

An iOS app powered by artificial intelligence.

## Overview

DreamAI is an iOS application that leverages AI capabilities to deliver intelligent features and experiences.

## Requirements

- iOS 26.0+
- Xcode 26.0+
- Swift 6.0+

## Getting Started

1. Clone the repository
2. Open `DreamAI.xcodeproj` in Xcode
3. Build and run the project
